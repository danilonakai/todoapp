Criando novo projeto ReactJS.
- Instalar o NodeJS
- Instalar o yarn (npm install --global yarn)
- Criar projeto com vite (yarn create vite) (Configurar projeto pelo prompt)
- Entrar na pasta do projeto e instalar os "node_modules" (yarn)
- Rodar o projeto (yarn dev)
- Alterar estrutura do projeto (Criar pasta "assets" dentro da pasta src, dentro de assets criar as pastas "css" e "images")
- Dentro da pasta src criar a pasta "components"
- A cada component sera criado uma pasta com a primeira letra maiuscula e dentro dessa pasta vai ter o component e o css com a primeira letra maiuscula
